# TrustCard Frontend

Frontend prototype for the **TrustCard: AI-Powered Social Media-Based Credit Card Recommendation System**.

## Overview

The frontend demonstrates the user interface and user experience of the TrustCard platform, including:

* Trust score visualization
* Behavioral signal dashboard
* Credit card recommendation interface
* Privacy and fairness information
* Modern responsive design
